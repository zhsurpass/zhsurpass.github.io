JSON简介

JSON定义

JSON的全称是"JavaScript Object Notation"，意思是JavaScript对象表示法，它是一种基于文本，独
立于语言的轻量级数据交换格式。

JSON元素

JSON规则

  • 空值：null
  • 布尔：true、false
  • 数字：123，1.23
  • 字符串：由双引号包裹
  • 数组(Array)：用方括号[]表示，里面可以包含空值、布尔、数字、字符串、数组、对象
  • 对象(Object)：用大括号{}表示，里面由键值对组成，键是字符串，值可以是空值、数字、布尔、字符串、数组、对象
  • 并列的数据之间用逗号,进行分隔

JSON官网

https://www.json.org/json-zh.html
